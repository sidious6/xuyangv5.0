# 24节气与保健之道

## 立春（2月3-5日）
**养生要点**：养肝护阳，顺应春生
- **饮食调理**：宜食辛甘发散之品，如韭菜、葱、姜、蒜等，以助阳气生发
- **起居作息**：早睡早起，与日俱兴，避免熬夜
- **运动锻炼**：适宜散步、太极拳等和缓运动，避免剧烈运动
- **情志调节**：保持心情舒畅，避免怒气伤肝
